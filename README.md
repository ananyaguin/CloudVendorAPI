# Cloud Vendor REST API

🚀 Spring Boot backend project implementing RESTful CRUD APIs for Cloud Vendor Management using MySQL and layered architecture.

## 🛠️ Tech Stack
- Java
- Spring Boot
- MySQL
- Spring Data JPA
- Swagger/OpenAPI
- Maven
- Postman

## 📂 Architecture
- Controller Layer
- Service Layer
- Repository Layer
- Model Layer
- Exception Handling
- Custom API Responses
- Swagger Documentation
- Actuator Monitoring

## ✨ Features
✔ CRUD REST APIs  
✔ MySQL Integration  
✔ Swagger API Documentation  
✔ Global Exception Handling  
✔ Spring Boot Actuator Monitoring  
✔ Layered Architecture Implementation
